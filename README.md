# Portfolio

A modern, responsive personal portfolio built using React, Vite, and Tailwind CSS.
This portfolio showcases my skills, projects, and experience with a clean UI and fast performance.

⸻

🚀 Tech Stack
•	React – Component-based UI
•	Vite – Fast build tool & dev server
•	Tailwind CSS – Utility-first CSS framework
•	JavaScript (ES6+)
•	HTML5 & CSS3

⸻

✨ Features
•	Responsive design (mobile, tablet, desktop)
•	Clean and modern UI
•	Fast loading with Vite
•	Reusable React components
•	Easy to customize
•	SEO-friendly structure

# 🛠️ Installation & Setup

## ️⃣ Clone the repository
git clone https://github.com/MOHITGODARA1/portfolioo.git

## 2️⃣ Navigate to the project folder
cd portfolio

## 3️⃣ Install dependencies
npm install

## 4️⃣ Start the development server
npm run dev

## 🧪 Build for Production
npm run build
🎨 Customization
•	Update content in src/components
•	Change colors & styles in tailwind.config.js
•	Replace images in src/assets
•	Edit metadata in index.html
